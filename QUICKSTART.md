# Быстрый старт / Quick Start

## 1. Проверьте, установлен ли ffmpeg:
```bash
ffmpeg -version
```

Если нет, установите:
```bash
brew install ffmpeg
```

## 2. Установите зависимости Python:
```bash
pip install -r requirements.txt
```

## 3. Запустите приложение:
```bash
python app.py
```

## 4. Откройте в браузере:
```
http://localhost:5500
```

## Пример использования:

1. Вставьте ссылку на видео из VK:
   `https://vk.com/video-XXXXXX_XXXXXX`

2. Укажите время начала (по умолчанию 10 секунд)

3. Нажмите "Создать GIF"

4. Скачайте готовый GIF!

---

## Troubleshooting:

**Проблема:** "ffmpeg not found"
**Решение:** Установите ffmpeg через brew (macOS) или apt (Linux)

**Проблема:** Ошибка при скачивании с VK
**Решение:** Убедитесь, что видео доступно публично и не ограничено по регионам

**Проблема:** GIF не создается
**Решение:** Проверьте, что указанный тайминг не выходит за длительность видео
